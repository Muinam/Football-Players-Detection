# ⚽ Football Players Detection using YOLOv8

This project implements a Football Player Detection system using YOLOv8.  
It trains a custom object detection model on a Roboflow dataset and performs inference on both images and videos.

---

## 🚀 Project Overview

The system:

- Downloads a labeled football player dataset from Roboflow
- Trains a YOLOv8 model
- Performs object detection on validation images
- Runs inference on football match videos
- Displays detected players with bounding boxes

This project demonstrates practical Computer Vision skills using Ultralytics YOLOv8.

---

## 🛠️ Technologies Used

- Python
- YOLOv8 (Ultralytics)
- Roboflow
- OpenCV
- Jupyter Notebook
